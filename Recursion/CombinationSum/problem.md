# Combination Sum

## Problem Statement

Given an array of **distinct integers** `candidates` and an integer `target`, return **all unique combinations** of `candidates` where the chosen numbers sum to `target`.

Each number in `candidates` may be used an **unlimited number of times**.


---

## Constraints

- 1 ≤ len(candidates) ≤ 30  
- 1 ≤ candidates[i] ≤ 200  
- 1 ≤ target ≤ 500
